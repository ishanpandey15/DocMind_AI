# /vectordb folder

This folder stores the FAISS vector index after indexing.

After you click "Build Index" in the app, two files will appear here:

    index.faiss  → binary FAISS index file containing all embedded vectors
    index.pkl    → pickle file with chunk metadata (text, source, etc.)

These are auto-generated by LangChain's FAISS.save_local() method.
Do not manually edit these files.
